I haven’t received an email with my 4-digit code

This could be because:

  • The email can take a few minutes to arrive in your email inbox
  • The email may be in the spam / junk / trash folder of your email inbox
  • The email address that you entered on the previous page (emailjustentered@gmail.com) is not the email address that you used to save the application
  • It is more than XXX days since you saved your application
If you have followed the guidance above and have still not received a code, you will need to contact info here..
My 4-digit code doesn’t work

This could be because:

  • You have entered a postcode that doesn’t match the one on the application that you saved
  • You have entered a 4-digit code that doesn’t match the one on the email that you have just received
  • You have entered the postcode and the 4-digit code in the wrong boxes
  • The 4-digit code has expired, in which case you can request a new code
  • You have made 10 unsuccessful attempts to return to your application, and have been locked out for 5 minutes
If you have followed the guidance above and have still not been able to return to your application, you will need to start a new application.
